# Abstract

This report presents the design and implementation of a web-based Event Ticketing System that aimed to improve the efficiency of managing event operations and high-concurrency ticket bookings. The system supports multiple user roles, including customers, organizers, staff, and administrators, and provides core functionalities such as dynamic seat map management, real-time asynchronous ticket purchasing, secure E-ticket generation, and automated attendee check-in.

The application is developed using a microservices-based architecture with distributed event-driven data management and role-based access control to ensure data integrity and security. By enabling reliable high-volume transactions and real-time status updates, the proposed system enhances operational transparency, reduces manual administrative workload, and improves overall management efficiency within the event and entertainment industry.
